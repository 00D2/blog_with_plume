人工智能（Artificial intelligence）

机器学习是一种实现人工智能的方法；深度学习是一种实现机器学习的技术。
机器学习：使用算法来解析数据，从中学习，然后对真实世界中的事件做出决策和预测。比如：垃圾邮件检测、房价预测。
深度学习：模仿人类神经网络，建立模型，进行数据分析。比如：人脸识别、语义理解、无人驾驶。
