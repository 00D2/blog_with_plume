# Chrome浏览器任意修改网页内容

- 在Chrome浏览器按F12进入开发者模式，切换到Console选项卡
- 输入`document.body.contentEditable="true"`
- 直接在网页上修改对应的信息即可。

> 这个是利用了H5的contentEditable属性
